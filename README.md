# astrbot_plugin_quote_collocter_plus
回复抽象发言自动入典

> Fork 自 [litsum/astrbot_plugin_quote_collocter](https://github.com/litsum/astrbot_plugin_quote_collocter)

## 使用方法
发送"语录投稿+图片"或"入典+图片"，bot会将群u语录保存到本地（不同群聊相互隔离）  
如果你的qq未开启半屏相册发图功能，可以通过【qq左划→设置→通用→图片、视频→发图方式】进行修改  
权限系统可以通过发送"投稿权限+模式数字"进行设置  
   0：关闭投稿系统  
   1：仅管理员可投稿  
   2：全体成员均可投稿  

在bot被戳一戳时，将会随机选择一张黑历史发送（有10秒默认冷却）  
如果想要修改这个冷却，请发送"戳戳冷却+冷却时间"来设置，单位为秒  
发送"/语录"或"语录"可以主动触发随机语录  

## 引用文本转图片（自适应渲染）
- 引用纯文本投稿时，插件会按文本长度自动调整气泡宽度与图片尺寸。  
- 短文本不会被强制拉满，长文本会在上限内自动换行。  
- 对超长连续字符（如长 URL）启用强制断行，避免内容溢出。  
- 保留原始换行语义；纯空白文本会使用兜底文案，超长文本会截断。  

## 群相册上传（NapCat）
开启“投稿后自动上传群相册”后，插件会在语录图片本地保存成功时，额外调用 NapCat 群相册接口上传一份到目标相册。
本地保存仍然是主流程；群相册上传失败不会删除本地语录，也不会影响后续"/语录"和戳一戳随机发送。

配置方式：
- 在 AstrBot 管理面板中开启“投稿后自动上传群相册”
- 填写“目标群相册名称”或“目标群相册ID”
- 如果 AstrBot 与 NapCat 不在同一文件环境，建议保持“相册上传启用 Base64 兜底”为开启

群内管理员命令：
- `语录相册 状态`：查看当前群相册上传设置
- `语录相册 列表`：获取当前群可用相册及 ID
- `语录相册 开启` / `语录相册 关闭`：覆盖当前群上传开关
- `语录相册 名称 相册名`：设置当前群目标相册名称
- `语录相册 ID 相册ID`：设置当前群目标相册 ID
- `语录相册 严格开启` / `语录相册 严格关闭`：控制找不到相册名时是否严格停止上传
- `语录相册 重置`：恢复为插件全局配置

> 该功能依赖 NapCat/OneBot 扩展接口：`get_qun_album_list` 与 `upload_image_to_qun_album`。机器人账号需要具备对应群相册上传权限。

## 插件配置
插件已按 AstrBot 官方配置规范添加 `_conf_schema.json`，可在 AstrBot 管理面板中配置：
- 插件管理员 QQ 号列表，以及是否合并 AstrBot 全局管理员
- 语录图片保存目录
- 新群默认投稿权限
- 默认戳戳冷却时间
- 是否启用戳一戳随机语录、戳一戳发送语录概率
- 是否允许引用文本转图片投稿、文本投稿最大字符数
- 是否投稿后自动上传群相册、目标相册名称或 ID、上传兜底与结果提示

## 注意事项
图片投稿没有限制，请自己注意审核
