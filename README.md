# astrbot_plugin_quote_collocter_plus

回复抽象发言，自动入典。

> Fork 自 [litsum/astrbot_plugin_quote_collocter](https://github.com/litsum/astrbot_plugin_quote_collocter)

这是一个基于 AstrBot 的语录收集插件。  
除了保留原版的图片投稿能力外，这个增强版还补上了文字引用渲染、合并聊天记录转图，以及自动上传群相册等更适合日常群聊使用的能力。

## 相比原仓库有什么优点？

原版主要支持“投稿图片”这一种入典方式；这个版本在此基础上做了更完整的增强：

- 支持引用文字直接入典，不再只能投稿图片。
- 支持将合并转发 / 合并聊天消息自动渲染成聊天记录图片后保存。
- 支持投稿成功后自动上传到群相册，方便集中归档和后续查看。

简单说，就是从“只能收图”扩展成了“图片能收、文字能收、聊天记录也能收，而且还能顺手同步进群相册”。

## 主要功能

- 支持传统图片投稿。
- 支持引用一段纯文字并生成聊天气泡图投稿。
- 支持解析合并转发中的文字与图片，自动生成聊天记录图片。
- 投稿内容按群隔离保存，不同群互不影响。
- 支持 `/语录` 或 `语录` 随机抽取历史语录。
- 支持被戳一戳时随机发送一条语录，可配置冷却与触发概率。
- 支持 NapCat 群相册自动上传。

## 使用方法

### 1. 图片投稿

发送以下任一形式即可：

- `入典 + 图片`
- 回复一条消息后发送 `入典`
- 回复一张图片后发送 `入典`

插件会将语录图片保存到本地，并按群聊进行隔离。

### 2. 引用文字转图片投稿

引用一段纯文字后，发送：

- `语录投稿`
- `入典`

插件会自动把引用内容渲染成聊天气泡图片后保存。

渲染规则：

- 会根据文本长度自动调整气泡宽度与图片尺寸。
- 短文本不会被强行拉满，长文本会自动换行。
- 超长连续字符（如长链接）会强制断行，避免溢出。
- 保留原始换行语义。
- 纯空白文本会使用兜底文案，超长文本会按配置截断。

### 3. 合并聊天消息自动转图投稿

引用合并转发 / 合并聊天消息后发送：

- `入典`

插件会把每个转发节点中的文字和图片合成为一张聊天记录图片并保存。

说明：

- 会处理节点中的文字和图片内容。
- 视频、文件、语音等其他内容会被忽略。

### 4. 随机语录

- 发送 `/语录` 或 `语录` 可主动随机获取一条历史语录。
- Bot 被戳一戳时，也可以随机发送一张语录图。

## 群相册自动上传（NapCat）

开启“投稿后自动上传群相册”后，插件会在本地保存语录图片成功后，额外调用 NapCat 群相册接口上传一份到目标相册。


### 配置方式

- 在 AstrBot 管理面板中开启“投稿后自动上传群相册”。
- 在“目标群相册名称列表”中按 `群号:群相册名` 配置，例如 `123456:黑历史`。
- 如果你想固定使用某个相册 ID，也可以直接填写“目标群相册ID”。
- 如果 AstrBot 与 NapCat 不在同一文件环境，建议开启“相册上传启用 Base64 兜底”。

### 群内管理命令

- `语录相册 状态`：查看当前群相册上传设置
- `语录相册 列表`：获取当前群可用相册及 ID
- `语录相册 开启` / `语录相册 关闭`：覆盖当前群上传开关
- `语录相册 名称 相册名`：设置当前群目标相册名称
- `语录相册 ID 相册ID`：设置当前群目标相册 ID
- `语录相册 重置`：恢复为插件全局配置

> 该功能依赖 NapCat / OneBot 扩展接口：`get_qun_album_list` 与 `upload_image_to_qun_album`。机器人账号需要具备对应群相册上传权限。

## 权限相关

可通过群内命令设置投稿权限：

- `投稿权限 0`：关闭投稿系统
- `投稿权限 1`：仅管理员可投稿
- `投稿权限 2`：全体成员均可投稿

也可以设置戳一戳冷却时间：

- `戳戳冷却 + 秒数`

## 注意事项

- 图片投稿本身不做内容审核，请自行注意使用场景。
- 如果你的 QQ 未开启半屏相册发图，可能会影响相关体验，可在 QQ 设置中调整发图方式。
